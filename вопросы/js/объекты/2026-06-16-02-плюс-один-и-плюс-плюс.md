## Почему `obj.age + 1` не работает, а `obj.age++` - работает?

Было задание: прибавить 1 к возрасту в объекте и вернуть объект. Я написал так:

```js
function increaseAge(obj) {
    obj.age + 1
    return obj
}

console.log(increaseAge({name: "Анна", age: 25}))  // {name: "Анна", age: 25}
```

Но это не сработало, почему?

Потому что `obj.age + 1` вычисляет новое значение, но никуда его не сохраняет. `obj.age` остаётся прежним

Есть 2 правильных варианта:

**1 вариант:** `obj.age = obj.age + 1` - просто сохраняем результат обратно

**2 вариант:** `obj.age++` - сокращённая запись

Оба работают:

```js
function increaseAge(obj) {
    obj.age = obj.age + 1  // или obj.age++
    return obj
}

console.log(increaseAge({name: "Анна", age: 25}))  // {name: "Анна", age: 26}
```

**В чём разница между `++` и `+ 1` если оба прибавляют 1?**

`++` - это **команда**, которая меняет `obj.age`. Увеличивает на 1 и сохраняет результат обратно в переменную

`+ 1` - это **выражение**, которое вычисляет новое значение (старое + 1), но никуда не сохраняет. `obj.age` остаётся прежним
