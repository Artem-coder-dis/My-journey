## Как поменять местами значения двух ключей в объекте

Было задание: поменять местами значения двух ключей и вернуть обновлённый объект

Я написал:

```js
function swapKeys(obj, key1, key2) {
    let temp = Object.keys(obj)
    temp[0] = temp[1]
    
    return obj
}

console.log(swapKeys({name: "Анна", age: 25}, "name", "age")) // {name: 'Анна', age: 25}
```

**Какие ошибки были совершены:**

1. Не использованы параметры `key1` и `key2`
2. Я менял элемент в массиве `temp`, но это массив ключей, не сам объект. Значения в объекте не меняются

**Правильно:**

```js
function swapKeys(obj, key1, key2) {
    let temp = obj[key1]
    obj[key1] = obj[key2]
    obj[key2] = temp
    
    return obj
}

console.log(swapKeys({name: "Анна", age: 25}, "name", "age")) // {name: 25, age: 'Анна'}
```

**Что мы сделали?**

Сохранили значение первого ключа во временную переменную, первому ключу присвоили значение второго, второму - сохранённое значение первого. Обмен местами, как с двумя переменными

**Вывод:** `Object.keys(obj)` создаёт новый массив с копиями ключей. Изменения в этом массиве не влияют на сам объект

Обмен значений через временную переменную - это приём, который мы будем часто использовать в дальнейшем
